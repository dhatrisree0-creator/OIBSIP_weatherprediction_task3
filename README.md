# OIBSIP Python Programming Task - Weather Prediction App

## Objective

The objective of this project is to develop a Weather Prediction application that allows users to check real-time weather information for any city. The application fetches live weather data and displays important weather metrics through an interactive graphical user interface.

## Description

This project is developed using Python, Tkinter, and the OpenWeatherMap API. Users can enter a city name and instantly retrieve current weather details such as temperature, humidity, weather condition, and description. The application features a modern and dynamic user interface that changes colors based on weather conditions to improve user experience.

## Features

* Live weather information retrieval
* Search weather by city name
* Displays:

  * Temperature
  * Humidity
  * Weather Condition
  * Weather Description
  * Location Information
* Dynamic weather-themed color changes
* User-friendly graphical interface
* Error handling for invalid city names and network issues

## Tools & Technologies Used

* Python 3
* Tkinter (GUI Library)
* Requests Library
* OpenWeatherMap API

## Steps Performed

1. Designed the graphical user interface using Tkinter.
2. Integrated the OpenWeatherMap API for live weather data.
3. Created an input field for city name entry.
4. Implemented API requests using the Requests library.
5. Extracted temperature, humidity, and weather condition data.
6. Added dynamic color themes based on weather conditions.
7. Implemented error handling for invalid inputs and network failures.
8. Tested the application with multiple city names.

## Output

The application fetches and displays live weather information, including temperature, humidity, weather condition, and location details in a visually appealing weather dashboard.
